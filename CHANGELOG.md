# 📋 KeidoNav 修改日志 (Changelog)

> 本项目采用 **Keep a Changelog** 格式记录所有重要修改。  
> 每次提交后会自动推送到 GitHub。若需手动推送，请运行 `git push origin main`。

---

## 格式说明

- `feat:` — 新功能（feature）
- `fix:` — 修复 Bug
- `perf:` — 性能优化
- `refactor:` — 代码重构（不影响功能）
- `docs:` — 文档更新（README、CHANGELOG 等）
- `style:` — 代码格式调整（不影响逻辑）
- `chore:` — 构建/工具链/依赖等杂项
